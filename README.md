# GremlinConverter
Converts Gremlins from [KurtVelasco's Desktop Gremlin](https://github.com/KurtVelasco/Desktop_Gremlin) on Windows to [linux-desktop-gremlin](https://github.com/iluvgirlswithglasses/linux-desktop-gremlin) by @iluvgirlswithglasses
